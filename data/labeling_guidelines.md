## Action item definition

An action item is:
A concrete task someone is responsible for completing in the future.

## Include:

“I’ll…”

“Can you…”

“Let’s make sure to…”

“We need to…”

## Exclude:

Opinions

Decisions without tasks

Status updates

## Owner rules

If speaker says “I’ll” → that speaker

If name mentioned → that name

If “we” → leave owner as "unspecified"

## Deadline rules

Explicit dates → keep as is

Relative time (“next week”) → keep text

No deadline → "unspecified"

Never hallucinate missing info.
